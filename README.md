# khuthon-app

A Flutter + FastAPI + Supabase hackathon MVP that helps users rethink overconsumption by comparing eating-out costs with home-cooking costs.

## Stack

- Flutter
- FastAPI
- Supabase PostgreSQL
- LLM API

## Structure

khuthon-app/
- docs/
- app/
- backend/

## Branches

- main: final stable demo branch
- frontend: Flutter app development
- backend: FastAPI + Supabase + LLM development
